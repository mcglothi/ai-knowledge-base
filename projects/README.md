## Adding a project

```bash
cp _templates/file-template.md projects/your-project-name.md
```

Fill in the template, then add a row to `_index.md`:
```markdown
| Project Name | 🟢 Active | tag1, tag2, python | [`projects/your-project-name.md`](projects/your-project-name.md) |
```

See `example/projects/my-project.md` for a complete example.
