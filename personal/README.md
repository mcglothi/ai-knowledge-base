## Getting started

1. Copy `../../example/personal/profile.md` → `profile.md` and fill it in
2. Create `dev-environment/` directory
3. Copy `../../example/personal/dev-environment.md` → `dev-environment/README.md`
4. For each machine, copy `../../_templates/machine-profile.md` → `dev-environment/<hostname>.md`

These two files are the most important ones to fill in first — agents load them at the start of every relevant session.
